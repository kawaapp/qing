# 卡哇轻论坛

该论坛系统源自于 [卡哇微社区](https://kawaapp.com) 的后端核心系统，剥离了卡哇业务业务相关的模块，仅留下最核心的社区模块。
论坛采用前后端分离架构，前端采用 React 打造，后端服务用 Go. 

当前开源的部分是后端模块。

该项目原来的前端是微社区小程序，这个项目是按PC论坛的风格重新设计的，原来的小程序API不兼容所以没有前端，需要前端的请绕过。
